PDF Hosting for AI Testing

A simple page to host and access PDF files for your AI backend projects.

How to Use This Page

  1. Create a folder named `pdfs` in your GitHub repository.
  2. Upload your PDF files into that new `pdfs` folder.
  3. Edit this `index.html` file and add links to your files in the list below, using the format: ./pdfs/your-file-name.pdf
  4. Push the changes to GitHub, and your files will be live!
